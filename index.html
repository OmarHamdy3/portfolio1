Autonomous Driving Perception System

Project Description

This project developed a real-time perception system for autonomous vehicles using state-of-the-art deep learning models. The system processes camera, LiDAR, and radar data to detect, classify, and track objects in complex urban environments.

The architecture combines YOLOv8 for object detection with DeepLabV3+ for semantic segmentation, achieving real-time performance (30 FPS) on embedded NVIDIA hardware. The system was validated on 50+ hours of driving data across diverse weather conditions.

Medical Image Diagnosis AI System

Project Description

This project developed a deep learning system for automated diagnosis of medical images, specifically focusing on detecting abnormalities in chest X-rays and brain MRIs. The system assists radiologists by providing second opinions and prioritizing urgent cases.

Using a combination of convolutional neural networks and vision transformers, the system achieved radiologist-level accuracy while reducing diagnosis time from 15 minutes to 30 seconds per scan. The project involved collaboration with two major hospitals and was validated on 10,000+ anonymized patient scans.
